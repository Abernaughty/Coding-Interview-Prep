# Coding Interview Prep

Python implementations and exercises for common coding interview topics.

## Topics Covered

### Arrays (`Arrays/`)
| File | Description |
|---|---|
| `Implementation.py` | Core array operation implementations |
| `Interview_Questions.py` | Common array interview questions with solutions |
| `Merge_sorted_arrays.py` | Merge two sorted arrays into one |
| `Reverse_string.py` | String/array reversal approaches |

### Hash Tables (`main.py`)
Custom `HashTable` class implementation with:
- `set(key, value)` — insert with hash-based placement
- `get(key)` — retrieve by key
- `delete(key)` — remove entry
- Collision handling via chaining

### Problem-Solving Methodology (`How To Solve Coding Problems/`)
| File | Description |
|---|---|
| `google_interview.py` | Walkthrough of the Google interview problem-solving framework |
| `interview_question_1.py` | Worked example applying the methodology |

## Running

No dependencies — standard Python 3.

```bash
python main.py
python Arrays/Implementation.py
```
